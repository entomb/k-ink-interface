What is k(InK)

k(Ink) is a kinky way to interact with the ink.js library easing the pain of designers and developers alike when using the awesome Ink css framework.

k(Ink) takes some of the functionality of the to the full fledged ink.js and creates a set of chainable methods and aliases, allowing for easy usage, fast implementation and code readability. The library comes in a small <10kb minified file that works with the most recent Ink (v2) package.
